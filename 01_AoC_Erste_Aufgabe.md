# Advent of Code – dein erstes Mini-Puzzle

*Marktstand · Claude Code · Übung 1 von 5*

**~ 10 Min · für absolute Anfänger:innen · Du brauchst: Neugier**

Hier ist eine kleine Knobelaufgabe – direkt im Handout. Kein Browser nötig. Lass Claude sie zusammen mit dir lösen, in der Sprache, die dich interessiert.

> **Dein Prompt**
>
> Hier ist eine kleine Knobelaufgabe: Für jede Zeile finde die erste und die letzte Ziffer, kombiniere sie zu einer zweistelligen Zahl, summiere alles. Eingabe:

  1abc2
  pqr3stu8vwx
  a1b2c3d4e5f
  treb7uchet

Das Ergebnis muss 142 sein (12 + 38 + 15 + 77). Lös das in Python: speicher den Code als Datei, führ ihn aus, zeig mir das Ergebnis. Erklär mir die Lösung Schritt für Schritt – ich bin Anfänger:in.

## So gehst du vor

**Schritt 1 – Aufgabe lesen**

Die Aufgabe steht im Prompt. Lies sie zuerst selbst durch und überleg: was wäre die Antwort? Sie ist 142 – aber wie kommt man dahin?

**Schritt 2 – Lass Claude die Lösung schreiben**

Wähl eine Sprache (Python, Rust, TypeScript, was auch immer). Claude schreibt den Code, speichert ihn als Datei und führt ihn aus.

**Schritt 3 – Lass dir die Lösung erklären**

„Erklär mir deine Lösung Schritt für Schritt – als wäre ich Anfänger:in.“ Was du nicht verstehst, frag direkt nochmal nach.

**Schritt 4 – Wechsle die Sprache**

„Schreib das jetzt nochmal in einer anderen Sprache.“ Vergleich, was sich ändert. Lass dir die 3 spannendsten Unterschiede zeigen.

**Schritt 5 – Lass Unit Tests schreiben**

„Schreib 3 Unit Tests: die Beispiel-Eingabe muss 142 ergeben, eine Zeile ohne Ziffern gibt 0, eine Zeile mit nur einer Ziffer gibt diese doppelt. Führ die Tests aus.“

**Schritt 6 – README dazu schreiben**

„Erzeug ein README.md mit der Aufgabe in deinen Worten, wie man das Skript startet und wie man die Tests laufen lässt. Höchstens eine halbe Seite.“

## Wenn du noch Zeit hast

- Lass die Lösung in 3 Sprachen nebeneinander zeigen und vergleichen.
- „Was wäre der eleganteste, was der unleserlichste Weg, diese Aufgabe zu lösen?“
- „Welche Sprache fändst du für so eine Aufgabe am besten geeignet und warum?“
